# NexGear - Second-Hand Marketplace

This is a premium, modern, red-black gaming-themed second-hand marketplace website. It features a complete Node.js/Express backend and a Vanilla JS/CSS frontend with a beautiful user interface.

## Prerequisites
- Node.js installed

## Quick Start
1. Open terminal in the `marketplace` folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
4. Open your browser and go to: **http://localhost:3000**

## Admin Access
When the server runs for the first time, an admin user is automatically created.
- **Username**: admin
- **Password**: admin123

Login with this account and a special **"Command Center"** tab will appear where you can delete any listing.

## Features Included
- Beautiful Dark/Red aesthetic with glowing animations
- Semantic HTML5 and responsive Flexbox/Grid layouts
- Secure JWT-based Authentication
- Image uplodas using Multer (product pictures and avatars)
- Auto-created SQLite database (no separate database server needed)
- Full API routes for Users, Products, and Orders
- Admin Role capabilities
